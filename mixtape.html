How it works

  1. Upload or sync the latest masters to songs/mixtape/ and run ./scripts/upload_mixtape.sh.
  2. Run b2 file-url MAINMIXTAPE your-track.mp3 for every file to get the streaming links.
  3. Paste the URLs (and any metadata like durations) into scripts/mixtape-tracks.js.
  4. Deploy the site. This page pulls the config and starts streaming from Backblaze instantly.

Because playback happens straight from Backblaze, you only have to update the track config when the mixtape changes.
